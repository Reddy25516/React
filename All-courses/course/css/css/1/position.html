An element with position: static; is not positioned in any special way; it is always positioned according to the normal flow of the page An element with position: relative; is positioned relative to its normal position. it is not defined based on the previous or any other element The element is removed from the normal document flow, and no space is created for the element in the page layout. It is positioned relative to its closest positioned ancestor, if any; otherwise, it is placed relative to the initial containing block. An element with position: absolute; is positioned relative parent element what ever the we given position absolute then it will go for the parent element and position the current element based on it The element is positioned according to the normal flow of the document, and then offset relative to its nearest scrolling ancestor here it will be sticky to the top until the parent div is in the window once the parent div is out of window it will hide for it run these below code and see the result
